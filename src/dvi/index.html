S21_3DVIEWER

О программе

Данная программа является средством для просмотра 3D моделей в каркасном виде. Сами модели загружаются из файлов формата .obj. Просмотр моделей осуществляется с возможностями вращения, перемещения и масштабирования.

Установка

Для установки вам требуется использовать команду: make install. Для открытия программы перейдите в папку build в директории src и откройте приложение.

Удаление

Для удаления программы вы можете использовать команду make uninstall.
